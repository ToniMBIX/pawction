# Frontend (Vite + React + Tailwind)

Rutas:
- `/` inicio (destacados)
- `/auctions` listado
- `/auctions/:id` detalle + pujas (contador 24h)
- `/favorites` favoritos por usuario
- `/profile` perfil
- `/checkout/:id` pago del ganador

El cliente usa fetch hacia el backend Laravel (ruta base en `VITE_API_URL`).
